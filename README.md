# [Work in progress] AEON Sketchbook

AEON Sketchbook is a multi-platform application for designing and analysing large-scale logical models.

## Development

Make sure you have Rust and NPM installed. Then, after cloning, run `npm install` to download all JS/TS dependencies. To start the application in debug mode, run `npm run tauri dev`. To build a release version of the app, run `npm run build`.
